WordNet

Create a Simple network of words related to each other using Twitter Streaming API.

New

This is a open module I have developed for the soul purpose of creating network of contextually relative words, basically a simple form of word-play written in core python. As of now, this project includes an twitter streaming api helping to generate proper corpus for the network.

It uses TF-IDF values for words in given corpus as feature for a strong relation between words. My future goals for this module is to integrate more / better features for relation generation and more sources of corpus like facebook and reddit.

I am also planning to use graph-based databases for future integrations due to their similarity with the data I am generating.
